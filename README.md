# Open Agent Benchmark (OAB)

> An open-source benchmark for evaluating Large Language Models (LLMs), AI Agents, Retrieval-Augmented Generation (RAG) systems, Machine Learning assistants, and Software Engineering copilots on realistic, industry-oriented tasks.

---

## 📖 Overview

Open Agent Benchmark (OAB) is an open-source project that aims to provide a comprehensive, extensible, and reproducible benchmark for evaluating modern AI systems.

Unlike traditional prompt collections, OAB is designed around complete benchmark tasks that include metadata, evaluation rubrics, and reference solutions. The benchmark focuses on realistic scenarios encountered in software engineering, data science, business analytics, cloud computing, and AI development.

The long-term objective is to create a high-quality benchmark that can be used by researchers, developers, educators, and organizations to evaluate AI systems consistently.

---

## 🎯 Project Goals

* Build an open-source benchmark for evaluating AI systems.
* Cover multiple AI and Data Science domains.
* Support evaluation of:

  * Large Language Models (LLMs)
  * AI Agents
  * Retrieval-Augmented Generation (RAG) systems
  * Machine Learning assistants
  * Software Engineering copilots
* Provide structured metadata and evaluation rubrics.
* Maintain transparent documentation and version history.
* Publish benchmark releases for GitHub and Kaggle.
