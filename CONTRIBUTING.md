# Contributing to Open Agent Benchmark (OAB)

Thank you for your interest in contributing to Open Agent Benchmark!

We welcome contributions that improve the benchmark, documentation, tooling, and overall project quality.

## Ways to Contribute

* Report bugs
* Suggest new benchmark tasks
* Improve documentation
* Improve evaluation rubrics
* Add validation scripts
* Improve dataset quality
* Submit pull requests

## Development Guidelines

* Write clear and well-documented code.
* Follow the existing project structure.
* Keep commits focused on a single change.
* Test your changes before submitting.

## Commit Messages

We recommend using Conventional Commits.

Examples:

* `docs: update README`
* `feat: add benchmark generator`
* `fix: correct metadata validation`
* `refactor: simplify exporter`

## Pull Requests

Before submitting a pull request:

* Ensure your changes are tested.
* Update documentation if necessary.
* Keep pull requests focused and easy to review.

Thank you for helping improve Open Agent Benchmark!
